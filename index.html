hellojs-android

This application provides an interface to running JavaScript with SpiderMonkey on Android

  • Place your scripts (.js files) in the assets folder
  • Build the app
  • Install HelloJS.apk
  • Launch the SpiderMonkey HelloJS application
  • The first screen will display version info about the SpiderMonkey VM and bindings
  • Click 'Scripts' to display a list of JavaScript files that was placed in assets
  • Select the script you want to run
  • The return value of the scripts will be displayed in adb's logcat

Build

Prerequisites
  • spidermonkey
  • Android SDK
  • Android NDK
> cd hellojs-android
> ant clean && rm -rf obj && rm -rf libs && rm -rf bin
> NDK_ROOT=/android-ndk MOZILLA_ROOT=~/ws/mozilla ./build_native.sh
> ant debug
